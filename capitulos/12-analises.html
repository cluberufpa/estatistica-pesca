12  Análise: o exemplo do teste-t

Com os pressupostos checados, escolhemos a análise. A pergunta e o delineamento decidem — não o contrário. Neste capítulo trabalhamos um exemplo completo com o teste-t, a análise mais simples para comparar grupos. Ele é o nosso fio condutor: você verá o mesmo exemplo planejado (Pilar 1), analisado aqui (Pilar 2) e comunicado em PDF (Pilar 3). As demais técnicas — ANOVA, regressão, não-paramétrica, multivariada — ganham capítulos próprios mais adiante.

12.1 Mapa de decisão (visão geral)

Que análise usar? Uma prévia do que vem nos capítulos de técnica:

  • Comparar a média de 2 grupos → teste-t (nosso exemplo aqui).
  • Comparar a média de 3+ grupos → ANOVA.
  • Relação entre duas variáveis numéricas → regressão.
  • Quando a normalidade falha → alternativas não-paramétricas (Mann-Whitney, etc.).
  • Muitas variáveis ao mesmo tempo → multivariada (PCA, agrupamento).

Por enquanto, foquemos no caso de dois grupos.

12.2 A pergunta do exemplo

O peso das tainhas difere entre dois estuários (A e B)?

É uma comparação de uma variável numérica (peso) entre dois grupos (estuário A vs B). O teste indicado é o teste-t para duas amostras independentes — desde que os pressupostos sejam atendidos.

12.3 Passo a passo no R

library(tidyverse)
peixes <- read_csv("dados/desafio01_estuarios.csv")
peixes$estuario <- factor(peixes$estuario)

# 1. Descritiva por grupo — sempre olhe os dados primeiro
peixes |>
  group_by(estuario) |>
  summarise(n = n(), media = mean(peso_g), dp = sd(peso_g))

A fórmula peso_g ~ estuario lê-se “peso_g em função de estuario”: à esquerda a variável resposta, à direita o grupo. Essa gramática com ~ aparece em quase toda análise do R.

12.4 Checar os pressupostos (antes do teste!)

O teste-t assume independência, normalidade dentro de cada grupo e variâncias homogêneas. Reveja Capítulo 11 para o detalhe.

# Normalidade dentro de cada grupo (gráfico decide melhor que o p-valor)
peixes |> filter(estuario == "A") |> pull(peso_g) |> shapiro.test()
peixes |> filter(estuario == "B") |> pull(peso_g) |> shapiro.test()

# Homogeneidade de variâncias
library(car)
leveneTest(peso_g ~ estuario, data = peixes)

A IA roda t.test() sem nunca perguntar se os pressupostos foram atendidos. Esse é o seu trabalho. Se a normalidade falhar, a alternativa é o teste de Mann-Whitney (wilcox.test()) — e quem decide trocar é você, lendo os gráficos, não a IA.

12.5 Rodar o teste

# pressupostos OK → teste-t
t.test(peso_g ~ estuario, data = peixes)

# se a normalidade falhou → alternativa não-paramétrica
wilcox.test(peso_g ~ estuario, data = peixes)

12.6 Como reportar

Um resultado completo traz a estatística t, os graus de liberdade, o p-valor e a diferença entre as médias em gramas (o tamanho do efeito, que importa tanto quanto a significância). Exemplo de frase para o relatório:

“O peso médio diferiu entre os estuários (t = 2,43; gl = 67; p = 0,018), sendo cerca de 25 g maior no estuário A.”

No próximo capítulo (Capítulo 13), pegamos exatamente este resultado e o transformamos num relatório PDF — fechando o ciclo dos três pilares com um único exemplo.
